# Releases and beta versions

PR beta releases use the `beta` dist-tag. Stable releases are tag based and must not be published without explicit approval. The first stable package version is `0.1.0`.

## Francais

Les betas de PR utilisent le dist-tag `beta`. Les versions stables sont basees sur des tags et demandent une validation explicite. La premiere version stable du package est `0.1.0`.
